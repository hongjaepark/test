# 🧑‍💻 개발자 포트폴리오 (Developer Portfolio)

간단하고 직관적인 웹 개발자 포트폴리오 페이지입니다. 
저의 소개, 다룰 수 있는 기술 스택, 그리고 연락처 정보를 제공합니다.

## ✨ 주요 기능 (Features)

- **인터랙티브 UI**: 
  - 헤더 텍스트 타이핑 애니메이션 효과
  - 스크롤 시 부드럽게 나타나는 섹션 등장(Reveal) 애니메이션 (Intersection Observer 활용)
- **유틸리티 스크립트**: 개발자 도구(F12) 콘솔 창에서 확인할 수 있는 로또 번호 생성기 포함 (`main.js`)

## 🛠️ 기술 스택 (Tech Stack)

- **HTML5**: 웹 페이지 구조 작성
- **CSS3**: 스타일링 및 트랜지션 효과 구현
- **Vanilla JavaScript (ES6+)**: DOM 조작 및 인터랙티브 효과 구현

## 🚀 실행 방법 (How to Run)

별도의 서버 설정이나 설치 과정 없이, 다운로드 후 `index.html` 파일을 웹 브라우저(Chrome, Edge, Safari 등)에서 더블 클릭하여 열면 바로 확인할 수 있습니다.
