Azure OpenAI

엔터프라이즈급 보안과 성능을 제공하는 핵심 언어 모델 공급자

Azure OpenAI는 Microsoft Azure 클라우드 인프라 위에서 OpenAI의 강력한 언어 모델(GPT-4, GPT-3.5-Turbo 등)을 사용할 수 있게 해주는 서비스입니다.

1. 🛡️ 엔터프라이즈급 보안 및 규정 준수

일반적인 퍼블릭 OpenAI API와 달리, 기업 환경에 맞춘 보안을 제공합니다.

  • 고객이 입력한 프롬프트(데이터)가 모델 학습에 사용되지 않도록 보장합니다.
  • 금융 데이터나 개인 정보(세무 정보 등)를 다뤄야 하는 future-planner 프로젝트 특성상 보안이 매우 중요하므로 채택되었습니다.

2. 🚀 안정적인 성능 (SLA 보장)

Azure의 글로벌 인프라를 활용하여 높은 가용성과 빠른 응답 속도를 제공합니다.

  • 엔터프라이즈 트래픽에 맞춰 할당량을 조절하고, 갑작스러운 트래픽 증가에도 안정적인 서비스를 유지할 수 있습니다.

3. 💡 future-planner 프로젝트에서의 역할

이 프로젝트에서 Azure OpenAI는 다음과 같은 형태로 에이전트들의 핵심 지능(두뇌) 역할을 수행합니다.

  • Market Analyzer: 사용자의 성향을 분석하여 최적의 ETF 비율을 계산해내는 추론 능력.
  • Tax Expert: 어려운 세법 문서를 읽고(RAG) 문맥을 파악하여 절세 방안을 찾아내는 독해 능력.
  • Planner Inspector: 여러 에이전트의 결과를 취합하여 사람에게 자연스러운 문장으로 피드백을 작성해 주는 생성 능력.

프로젝트의 .env 파일에 Azure 엔드포인트와 API 키를 설정하고, LangChain의 AzureChatOpenAI 클래스를 통해 이 강력한 지능을 애플리케이션에 연결하고 있습니다.
