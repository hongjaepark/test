JavaScript (ES6+)

웹 페이지에 동적인 동작과 로직을 추가하는 프로그래밍 언어입니다.

JavaScript 역할

JavaScript는 사용자 입력 처리, DOM 조작, 비동기 통신, 애니메이션과 같은 클라이언트 사이드 동작을 구현합니다. ES6 이후에는 모듈, 화살표 함수, 클래스 등 최신 문법을 통해 가독성과 유지보수를 개선합니다.

주요 기능

  • 페이지 요소를 동적으로 변경
  • 이벤트 기반 인터페이스 구현
  • 서버와 비동기 통신(AJAX, Fetch)
  • 클라이언트 상태 관리 및 로직 처리

ES6+ 주요 특징

  • 화살표 함수, 템플릿 리터럴
  • let/const 변수 선언
  • Promise, async/await 비동기 처리
  • 모듈 시스템 및 클래스
