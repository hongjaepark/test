LangChain

LLM(대형 언어 모델) 애플리케이션 개발을 위한 강력한 핵심 프레임워크

LangChain은 OpenAI의 GPT 같은 대형 언어 모델(LLM)을 활용하여 강력하고 복잡한 애플리케이션을 쉽게 구축할 수 있도록 돕는 파이썬/자바스크립트 기반의 프레임워크입니다.

현재 future-planner 프로젝트에서는 AI 에이전트들이 사용자 입력과 외부 데이터를 바탕으로 사고하고 답변을 생성하는 두뇌의 뼈대 역할을 하고 있습니다.

1. 📝 프롬프트 관리 (Prompt Templates)

단순히 텍스트를 LLM에 전달하는 것을 넘어, 상황에 맞는 동적인 프롬프트를 생성합니다.

  • prompts.py 활용: "시장 분석가", "세무 전문가" 등 각 에이전트에게 부여할 페르소나와 제약 조건을 템플릿화하여 관리합니다.
  • 사용자의 입력(투자 성향, 월 목표 금액 등)을 템플릿의 변수로 주입하여 완성된 형태의 프롬프트를 LLM에 전달합니다.

2. 🔌 LLM 모델 연동 (Model Integration)

다양한 언어 모델 공급자(OpenAI, Anthropic, 오픈소스 모델 등)와의 통신을 추상화하여 제공합니다.

  • langchain-openai: 이 프로젝트에서는 Azure OpenAI 또는 일반 OpenAI API와 쉽게 연결하기 위해 사용됩니다.
  • API 키 관리, 재시도(Retry) 로직, 비동기 호출 등 복잡한 네트워크 통신을 한 줄의 코드로 단순화합니다.

3. 🧩 Output Parsers (출력 파서)

LLM의 텍스트 응답을 애플리케이션이 이해할 수 있는 구조화된 데이터(JSON, Pydantic 객체)로 변환합니다.

  • 앞서 설명된 Pydantic과 결합하여, LLM이 반환한 응답이 정확한 규격(예: ETFItem, TaxStrategy)에 맞는지 검증하고 파싱하는 역할을 수행합니다.
