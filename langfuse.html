Langfuse (LLMOps & Tracing)

LLM 애플리케이션의 성능, 비용, 그리고 AI의 생각 과정을 추적하는 강력한 모니터링 도구

Langfuse는 LLM(대형 언어 모델)을 활용한 애플리케이션을 위해 설계된 오픈소스 LLMOps(LLM Operations) 플랫폼입니다. 복잡하게 얽힌 AI의 동작을 시각화하고 최적화하는 데 사용됩니다.

1. 🔍 Tracing (실행 과정 정밀 추적)

LLM 애플리케이션은 단순히 질문을 던지고 답을 받는 구조를 넘어, 여러 단계(Chain)를 거치게 됩니다.

  • Langfuse는 AI가 어떤 프롬프트를 받았고, 어떤 데이터를 검색(RAG)했으며, 얼마나 시간이 걸려 응답했는지를 트리(Tree) 구조로 시각화해 줍니다.
  • 개발자는 이 추적 데이터를 통해 어디서 병목(지연)이 발생했는지, 왜 AI가 엉뚱한 대답(환각)을 했는지 원인을 즉각적으로 파악할 수 있습니다.

2. 📊 비용(Token) 및 성능 모니터링

상용 LLM(OpenAI 등) API는 입력과 출력 토큰(글자 수) 단위로 비용이 청구됩니다.

  • 사용자 요청 1건당 발생한 정확한 토큰 사용량과 예상 비용을 자동으로 계산하고 대시보드에 기록합니다.
  • 응답 지연 시간(Latency)과 에러율을 추적하여 서비스의 품질을 관리할 수 있게 돕습니다.

3. 💡 future-planner 프로젝트에서의 역할

이 프로젝트는 단순한 챗봇이 아니라 여러 에이전트(시장 분석가, 세무 전문가 등)가 서로 대화하고 문서를 검색하는 복잡한 LangGraph 워크플로우를 가지고 있습니다.

  • Langfuse는 에이전트들이 주고받는 모든 데이터와 사고 과정(ReAct 패턴)을 낱낱이 기록하는 항공기의 '블랙박스' 역할을 수행합니다.
  • 개발 단계에서 프롬프트를 튜닝(Prompt Engineering)할 때, 이전 버전과 성능을 비교하고 최적의 결과를 내는 프롬프트를 찾는 데 핵심적인 기여를 합니다.

👉 즉, "언어(LLM) 데이터를 연결하고 통합하는 도구"라는 의미입니다.
