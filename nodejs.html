Node.js / Express

서버 사이드 애플리케이션과 API를 작성하는 자바스크립트 런타임 및 프레임워크입니다.

Node.js 역할

Node.js는 서버에서 JavaScript를 실행할 수 있는 런타임 환경입니다. 이벤트 기반, 논블로킹 I/O 모델로 빠르고 확장 가능한 서버를 구현합니다.

Express 역할

Express는 Node.js 위에서 동작하는 경량 웹 프레임워크입니다. 라우팅, 미들웨어, 요청/응답 처리 기능을 제공하여 API 서버를 쉽게 구성할 수 있습니다.

주요 활용

  • RESTful API 서버 개발
  • 서버 사이드 렌더링 및 백엔드 로직 구현
  • 데이터베이스 연동 및 인증 처리
  • 마이크로서비스 및 서버리스 백엔드 구성
