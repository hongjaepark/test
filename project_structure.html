프로젝트 디렉토리 구조

AI Bootcamp 프로젝트의 파일 및 폴더 구성을 안내합니다.

프로젝트는 크게 프론트엔드 UI를 담당하는 최상단 스크립트와, 백엔드 로직을 담당하는 src 디렉토리로 나뉘어 있습니다.

future-planner/
│
├── streamlit_app.py         # 🖥️ 메인 웹 애플리케이션 (사용자 UI 및 결과 시각화)
├── .env                     # 🔐 환경 변수 (Azure OpenAI API 키 등) - 코드상 유추
├── data/                    # 📄 사용자가 업로드한 PDF 파일(세법/금융 가이드) 저장소
├── faiss_index/             # 🗄️ 로컬 벡터 데이터베이스 (FAISS) 저장소
│
└── src/                     # ⚙️ 핵심 비즈니스 로직 및 AI 에이전트 코드
    ├── agents/              # 🤖 AI 에이전트 관련 모듈
    │   ├── experts.py       # 에이전트들의 행동 로직 (시장 분석, 세무 분석 등)
    │   ├── prompts.py       # 에이전트들에게 부여할 페르소나 및 프롬프트 템플릿
    │   └── schemas.py       # LLM 응답을 규격화하기 위한 Pydantic 데이터 구조
    │
    ├── graph/               # 🕸️ 워크플로우 제어
    │   └── workflow.py      # 에이전트들을 순서대로 연결하는 LangGraph 정의 (코드상 유추)
    │
    └── utils/               # 🛠️ 유틸리티 및 보조 도구
        ├── retriever.py     # RAG(검색 증강 생성)를 위한 문서 로드 및 임베딩 처리
        └── finance.py       # LLM의 환각(Hallucination)을 막는 금융 수학적 계산 및 검증 도구
