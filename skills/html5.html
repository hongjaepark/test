HTML5 / CSS3

웹 페이지 구조와 스타일을 정의하는 기본 기술입니다.

HTML5 역할

HTML5는 웹 문서의 구조와 콘텐츠를 표현합니다. 시맨틱 태그, 폼, 미디어 요소 등으로 접근성 있고 의미 있는 웹 페이지를 만들 수 있습니다.

CSS3 역할

CSS3는 웹 페이지 스타일을 정의하며 레이아웃, 색상, 애니메이션, 반응형 디자인을 담당합니다. CSS 변환, 그리드, 플렉스박스를 사용하면 다양한 화면 크기에서 디자인을 조절할 수 있습니다.

주요 활용

  • 웹 페이지 뼈대 작성 및 콘텐츠 구조화
  • 레이아웃 구성 및 반응형 디자인 구현
  • 폼, 표, 이미지, 비디오 등 멀티미디어 요소 배치
  • 스타일 일관성 유지 및 시각적 인터랙션 제공
